# Foreword

- GrGen originated spring 2003 at Universität Karlsruhe (IPD Goos), initially using a relational database for subgraph matching, then reimplemented with search plan driven matching
- First public release: GrGen v1.4 (foreword July 2007); ported from C to C# by Moritz Kroll
- Alternative patterns and subpatterns added by Edgar Jakumeit (master's thesis); continued as open source after university support ended (foreword July 2014)
- From v7.0: database-like query result processing, new MSAGL-based graph viewer/debugger; from v8.0: SQLite-based persistent graphs (foreword December 2025)
- Current version: v8.0
